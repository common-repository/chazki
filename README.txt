Plugin to connect Chazki's Shipping services with WooCommerce

== Description ==
With this plugin you can easily integrate your WooCommerce store with chazki. This new plugin uses the new chazky system.
Requires WordPress 6.0 and PHP 7.

If you have suggestions for a new add-on, feel free to email me at integraciones@chazki.com

== Installation ==
1. Install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use WooCommerce->Shipping->Chazki screen to configure the plugin
4. Fill the data with the Api-key provided by Chazki as well as the Sucursal, which is the branch code provided by Chazki.
5. In this configuration, the field Estado a procesar shows in which state of the order will send a notificattion to chazki.
6. Then, you proceed to configure the zones (Woocomerce->Shipping Zones) where you add the shipping method selecting Chazki.
7. When selecting the service only must select the service that is contracted with Chazki. Otherwise, the notification will be denied by chazki.
8. There is two options in the configuration of the shipping method that is to select the category and size of product. If you choose None, these extra conditions are not applied.
